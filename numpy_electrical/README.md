# NumPy Electrical Simulations

This folder contains Python scripts applying NumPy,Matplotlib to demonstrate basic electrical engineering concepts.

## Programs
- `ac_waveform.py` → Simulates an AC voltage signal and computes peak,peak to peak and angular frequecny.
-  'RLC series resonance.py -Simulates current magnitude in RLC circuit and plots the frequency response.
